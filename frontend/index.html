Dataset & Model

Understanding the data and machine learning approach

üìä

Dataset: Credit Card Fraud 2023

This model is trained on the Credit Card Fraud Detection 2023 dataset containing real anonymized transactions from European cardholders.

  • 568,630 total transactions
  • 29 features: V1-V28 (PCA transformed) + Amount
  • 0.17% fraud rate (highly imbalanced)
  • Features V1-V28 are principal components from PCA for privacy
  • Amount is the only non-transformed feature
ü§ñ

Model: XGBoost Classifier

XGBoost (Extreme Gradient Boosting) was chosen for its excellent performance on imbalanced tabular data.

  • SMOTE oversampling to handle class imbalance
  • Stratified K-Fold cross-validation
  • Hyperparameter tuning with GridSearchCV
  • Optimized for maximum recall (catch all fraud)
  • Fast inference: <10ms per prediction
